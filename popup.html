Quick start

Set up your queue

  1. Add one class card per course.
  2. Put unique numbers in priority order, one per line.
  3. Click Use on the class you want first.
  4. Open the real or mock registration page, reload it, click the input box, and use the hotkeys.
Ctrl + Shift + S paste Ctrl + Shift + A previous Ctrl + Shift + F next class

Tip: drag the overlay by its colored header and resize it from the bottom-right corner.
